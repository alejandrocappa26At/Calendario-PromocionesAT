# Apuesta Total Calendar 🎰

Calendario interactivo profesional para Febrero 2026 con diseño elegante y moderno.

## Características

✨ **Diseño Profesional**
- Tema oscuro con colores corporativos de Apuesta Total
- Color primario: #121212 (Negro profesional)
- Acentos en rojo vivido: #E74C3C
- Tipografía: Inter (sans-serif)

📱 **Responsivo**
- Se adapta a cualquier dispositivo (desktop, tablet, móvil)
- Interfaz completa sin necesidad de scroll
- Scroll inteligente solo donde sea necesario

🎯 **Funcionalidades**
- Resaltado del día actual (2 de febrero)
- Identificación de fines de semana
- Efectos visuales suaves con animaciones
- Interfaz intuitiva y fácil de usar

## Acceso

🌐 **Ver en línea**: [Apuesta Total Calendar](https://alejandrocappa26At.github.io/Calendario-interactivo-2026/)

## Instalación Local

```bash
git clone https://github.com/alejandrocappa26At/Calendario-interactivo-2026.git
cd Calendario-interactivo-2026
```

Luego abre el archivo `calendario_junio_2026.html` en tu navegador.
